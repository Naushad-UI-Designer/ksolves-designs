Odoo Apps Guidelines

In odoo description - Not supported css and supported css guideline :-
  • 1.backgound-images property not supported.
  • 2.If we want to give background color- so we should use background-color property and we should avoid background
  • 3.anchor link will not support.
  • 4.we don't need to use the parent container - because when we add our code it's added to the container.we can take the inner container.
  • 5.position will not work properly- like absolute, relative
  • 6.box-shadow will not work
  • 7.we should use classes as we can - bootstrap classes - like w-100 , d-flex, h-100, justify-content-center, align-items-center etc
  • 8.max-height will not support we can use height or min-height.
  • 9.list-style:none; will not work. Use list-unstyled class of bootstrap.
